### Description:
This folder encompasses all files for my Unsupervised Machine Learning Application Project. 
My app allows users to interact and experiment with two unsupervised machine learning models, Principal Component Analysis (PCA) and K-Means. 
The app allows users to upload their own dataset or use provided demo datasets and select which features they would like to use in their model. 


Then, observe how altering hyperparameters impacts model training and performance.
