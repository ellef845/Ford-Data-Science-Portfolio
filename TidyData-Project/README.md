### Description:
This folder encompasses all files for my Tidy Data Project. My notebook, "tidy_data.ipynb", cleans and visalizes data from the 2008 Summer Olympics. It alters the original dataset to reflect the key principles of a tidy data. It ensures that each variable has its own column, each observation forms its own row, and each type of observational unit forms its own table. The project also includes visualizations of the cleaned dataset. It also includes visualizations which show the number of medals won by each sex as well as the top five sports in which greatest amount of metals were won and in which the least amount of medals were won. The notebook finishes with a pivot-table which sums the total number of metals for each sport segregated by sex. This project exemplifies proficiency in cleaning datasets and preforming basic Exploratory Data Analysis (EDA) in Python. 

### Data: 
My notebook utilizes a Github dataset titled "2008 Summer Olympics" originally created by Giorgio Comai (OBCT/CCI) and adapted by David Smiley. The adapted dataset provides information about medalists from the 2008 games. It includes variables such as an individual's sex, name, the sport they participated in, and the type of medal they won.

### Directions on how to run the notebook: 
1. Download the TidyData-Project folder.
2. Make sure you have seaborn, matplotlib.pyplot, and pandas installed within your python environment.
3. Run the code for the entire page.
